# ETF analysis with web application - Challenge 07
*Challenge 10 notebook use crypto currency data stored in a csv file to be clean and prepare for predicting values that allow us to cluster tickers by their returns. The notebook us the Standar Scaler to scale the information, the Kmeans to cluster the data and the PCA to optimize the same data*

---

## Technologies

This is a jupityer lab tool witch runs in python 3.7 with the following packages:

* [pandas](https://pandas.pydata.org/) - For pandas.

* [hvplot](https://hvplot.holoviz.org/) - For plots.

* [sklearn](https://scikit-learn.org/stable/) - Web app.

---

## Installation Guide

In order to runn the tool you need to first install the following pacakges:

```python
    pip install pandas
    conda install -c pyviz hvplot geoviews
    pip install -U scikit-learn
   ```
